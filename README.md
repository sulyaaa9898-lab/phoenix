# Phoenix Center

Локальная программа для одного учебного центра (английский и китайский), которая работает на одном компьютере через локальный сервер и локальную SQLite-базу.

## Цель

Система построена вокруг корректных связей между сущностями:

1. преподаватели
2. тарифы
3. группы
4. расписание
5. ученики
6. оплаты
7. график и табель преподавателей

Главная логика: преподаватель и расписание для ученика определяются через выбранную группу, а не назначаются вручную в карточке ученика.

## Технологии

- Node.js + Express
  Простой локальный сервер, легко поддерживать в одном центре.
- SQLite (встроенный модуль node:sqlite)
  Локальная и устойчивая БД в одном файле, транзакции и WAL-режим.
- EJS
  Серверные страницы без избыточной сложности.

## База данных

Используются ключевые таблицы:

- teacher_profiles
  Карточки преподавателей: язык, уровни, доступность, статус.
- tariff_plans
  Отдельные тарифы: язык, длительность, цена, статус.
- study_groups
  Учебные группы: язык, уровень, преподаватель, лимит мест, статус.
- group_schedules
  Слоты расписания, привязанные к группе и преподавателю.
- student_profiles
  Карточки учеников: язык, уровень, тариф, группа, статус.
- payment_records
  Фиксация оплат по тарифу и периода обучения (start/end).

## Ключевые проверки

При сохранении ученика и выборе группы система проверяет:

- совпадение языка ученика и группы
- совпадение уровня ученика и группы
- активность группы
- наличие активного преподавателя у группы
- наличие активного расписания у группы
- наличие свободных мест
- активность тарифа
- совпадение языка тарифа с языком ученика

## Что подставляется автоматически

После выбора группы для ученика:

- преподаватель определяется через группу
- расписание подтягивается через группу
- дополнительного ручного назначения преподавателя в карточке ученика нет

При оплате:

- тариф берется из формы или карточки ученика
- период обучения рассчитывается автоматически из длительности тарифа

## Локальный запуск

Требования:

- Node.js 20+

Команды:

```bash
npm install
npm start
```

После запуска:

```text
http://localhost:3000
```

## Хранение данных

- Данные хранятся локально в файле data/phoenix-center.db
- Включен SQLite WAL-режим для стабильной повседневной работы
- При обычной эксплуатации и резервном копировании файла БД данные сохраняются 12 месяцев и дольше
